# @vanillajs/css-prop

vanillajs css prop

## Installation

```bash
npm install @vanillajs/css-prop
```

## Usage

```js
import { css } from '@vanillajs/css-prop';
```

## Legacy UMD Usage

```html
<script src="//unpkg.com/@vanillajs/css-prop"></script>
```

```js
const { css } = window.vanillajs.cssProp;
```
